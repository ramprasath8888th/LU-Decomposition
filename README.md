# LU Decomposition 

## AIM:
To write a program to find the LU Decomposition of a matrix.

## Equipments Required:
1. Hardware – PCs
2. Anaconda – Python 3.7 Installation / Moodle-Code Runner

## Algorithm
1. Import the required libraries such as NumPy and SciPy.
2. Create a matrix using the np.array() function.
3. Use the scipy.linalg.lu() function to decompose the matrix into Lower triangular matrix (L) and Upper triangular matrix (U).
4. Display the L matrix, U matrix, and LU Decomposition result using the print() function.
